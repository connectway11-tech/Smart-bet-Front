# flutter_mvp

Alicación Flutter minimal que replica el layout provisto en el mock (pantalla principal con tarjetas de juegos y accesos rápidos).

## Requisitos

- Flutter SDK (estable). Instala desde https://flutter.dev

## Cómo ejecutar

1. Abrir la terminal en la carpeta del proyecto (desde la raíz del repo):

```bash
cd /ruta/al/proyecto/flutter_mvp
```

2. Obtener dependencias:

```bash
flutter pub get
```

3. Ejecutar en un dispositivo o emulador:

```bash
flutter run
```

## Notas

- Se usó `google_fonts` para tipografía. Añade tus assets (logos) en `assets/images/` y actualiza los widgets para mostrar imágenes reales.
